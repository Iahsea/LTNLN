---
description: Rà soát code hiện tại và cập nhật trạng thái checkbox trong PLAN.md
---

Rà soát toàn bộ code trong `backend/` và `frontend/`, so với checklist trong `PLAN.md`:
1. Tính năng nào đã implement xong và chạy được → đánh dấu `[x]`.
2. Tính năng nào chưa làm → giữ `[ ]`.
3. Cập nhật luôn cột "Trạng thái" trong bảng API endpoint.
4. Tóm tắt cho tôi: đã xong bao nhiêu %, còn lại những gì.
